# space репозиторий
Содержит файл space.py

Для загрузки только файла `space.py` можно рекомендовать следующие варианты:
```bash
#!/usr/bin/env bash
# Загружаем файл в текющую папку с помощью `curl`
curl -fsSL https://raw.githubusercontent.com/shirinst/space/refs/heads/main/space.py > space.py
```

Для работы с проектом, можно выполнить клонирование:
```bash
#!/usr/bin/env bash
# Загружаем файл в текющую папку с помощью `curl`
git clone https://github.com/shirinst/space.git
```
